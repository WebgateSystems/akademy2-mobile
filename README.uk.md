# AKAdemy2.0 (Flutter)

Мови: [Polski](README.md) · [English](README.en.md) · Українська

Мобільний застосунок для навчання: модулі з відео, інфографікою та тестами, офлайн-доступ через завантаження контенту, авторизація за телефоном/ПІН.

## Основний функціонал
- Авторизація за телефоном і ПІН, автозаблокування після повернення з фону.
- Каталог модулів/предметів із картками контенту (відео, інфографіка/PDF, тест).
- Прев’ю відео: локальні/мережеві файли та YouTube, субтитри, індикатор прогресу.
- Завантаження модулів для офлайну (файли, постери, субтитри) з контролем цілісності.
- Перегляд PDF/зображень у діалогах, інтерактивне збільшення.
- Локалізація через `flutter_localizations`/`intl`, світла та темна тема.
- Орієнтація: увесь застосунок у портреті; діалоги відео/YouTube дозволяють альбомну і повертають портрет після закриття.

## Технічна архітектура
- Flutter 3.4+/Dart 3.4+ (див. `environment` у `pubspec.yaml`), залежності керуються `flutter pub`.
- Стан: Riverpod; навігація: GoRouter; мережа: Dio (+логер, retry); сховище: Isar + secure storage.
- Завантаження та офлайн: `DownloadManager` з перевіркою підпису контенту та метаданими.
- Відео: `video_player` для локальних/мережевих файлів, `youtube_player_flutter` для YouTube, `flutter_pdfview` для PDF.
- Конфігурація API: compile-time `--dart-define=API_BASE_URL=<url>` (використовується в `core/network/api.dart` для `baseUrl`/`baseUploadUrl`).

## Як запустити
1) Встановити залежності:
```bash
flutter pub get
```
2) Запустити (вказати API):
```bash
flutter run --dart-define=API_BASE_URL=https://test.akademy.edu.pl
```
3) Зібрати реліз:
```bash
flutter build apk --release --dart-define=API_BASE_URL=https://your.api
# або
flutter build ios --release --dart-define=API_BASE_URL=https://your.api
```

## Як користуватися
- Увійдіть за телефоном/ПІН; за потреби прив’яжіть школу.
- Виберіть предмет/модуль → відкрийте картку контенту:
  - відео: локальне/мережеве або YouTube; субтитри підхоплюються з локальних/мережевих файлів;
  - інфографіка/PDF: перегляд у діалозі, можливість збільшення/скролу;
  - тест: перехід на екран Quiz.
- Для офлайну натисніть іконку завантаження модуля; індикатор показує прогрес.
- Відео-діалоги відкриваються у ландшафті за потреби; після закриття орієнтація знову портретна.

## Структура проєкту (скорочено)
- `lib/app` — тема, роутер, entry `App`.
- `lib/core` — мережа (Dio), сховища (Isar/secure storage), завантаження, синк, утиліти.
- `lib/features` — екрани: auth/join, home/subjects/modules, videos, profile, splash.
- `assets/images`, `assets/fonts` — ресурси (лого, Rubik/Inter).
- `l10n` — локалізації.
